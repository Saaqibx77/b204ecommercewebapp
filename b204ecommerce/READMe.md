
b204 webapplication built using html, css, javascript, expressjs,large part of data storage is done using MongoDb and json files


#How to run the application

Node.js should be installed in the PC. 


1. copy the project folder 
2.unzip the folder
3.cd path/to/b204ecommerce
4.npm install
 node server.js
5.open the web browser
   
 
   http://localhost:3000


home page will be displayed
